# StatVis

A simulation project to explore probability distributions (M.Sc. Data Analytics @ RWTH).

## Project Log
- **Milestone 1:** Basic project structure initialized. HTML skeleton, base CSS variables, and JS entry point.
- **Milestone 2:** Implemented responsive canvas resizing (handling DPR) and a basic random-noise test loop to verify drawing performance.
- **Milestone 3:** Added interactive UI controls (sliders) to adjust simulation speed and batch sizes in real-time.
- **Next Steps:** Start sorting the random data into histogram bins to see the distribution curve form.
