# StatVis

A simulation project to explore probability distributions (M.Sc. Data Analytics @ RWTH).

## Project Log
- **Milestone 1:** Basic project structure initialized. HTML skeleton, base CSS variables, and JS entry point.
- **Milestone 2:** Implemented responsive canvas resizing (handling DPR) and a basic random-noise test loop to verify drawing performance.
- **Step 3:** Added interactive UI controls (sliders) to adjust simulation speed and batch sizes in real-time.
- **Step 4:** Implemented histogram binning logic. Core math (Central Limit Theorem sampling) is now visually plotting a curve.
- **Step 5:** Added visual polish. The histogram now uses a sleek linear gradient, features a baseline axis, and displays live numerical stats on the canvas to look more professional.
- **Next Steps:** Consider adding different distribution types or finalize project documentation.
